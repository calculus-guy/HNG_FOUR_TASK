# Shared Types and Interfaces

This directory contains common types, interfaces, and utilities shared across all microservices.

## Usage

Copy the contents of this folder to each service's `src/common` directory or create a shared npm package.

## Contents

- `interfaces/` - Common interfaces for responses, pagination, etc.
- `decorators/` - Custom NestJS decorators
- `filters/` - Exception filters
- `interceptors/` - Request/response interceptors
- `guards/` - Auth guards
- `utils/` - Helper functions
